BLUE TEX WEBSITE PRIORITIES:
The new site should push these main things:
1. Wall Cover & Repair System
2. Roof Cover & Repair System
3. Metal Building Retrofit Applications
4. Roll-up Door Applications

Everything else is secondary.

BLUE TEX POSITIONING:
BlueTex is for metal building owners who want to cover, repair, or retrofit old/damaged insulation and improve the finished interior look of their building.

The site should make it very easy to understand:
- what the system is
- where it installs
- what parts are used
- what the finished result looks like
- how to measure
- how to request a quote
- how to order the sample/test kit

VOICE:
Use short, direct copy.
Sound like a practical metal building insulation expert.
Avoid generic AI marketing language.
Avoid long em dashes.
Avoid “shop now.”
Use quote-focused CTAs.

PRIMARY CTA LANGUAGE:
- Request a Quote
- Send Photos for a Quote
- Get a Material List
- Order the $100 Test Kit
- View Install Guide
- View Application Overview

DESIGN STYLE:
Industrial, practical, clean, and trustworthy.
Do not make it look fake glossy.
Use metal building imagery and before/after proof.
Use BlueTex colors:
- Navy: #0b2f6b
- Orange: #ee7f12
- White
- Light gray
- Dark gray/black body text
